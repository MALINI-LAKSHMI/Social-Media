# Social-Media

### Project Description

This project is a console-based Social Media Application developed using Java following Low Level Design (LLD) principles.
The application simulates basic features of a real-world social media platform where users can create accounts, connect with friends, share posts, interact with posts, and receive notifications.

The system is designed using object-oriented programming concepts such as encapsulation, modular design, and separation of concerns with a clear package structure.

## Classes used:

User → stores user details, friends, posts, notifications

Post → stores post content, likes, comments

SocialMediaService → contains business logic

MainApp → runs the application
